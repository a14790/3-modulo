# 📘 3º Módulo – Exercícios de Python

Este repositório contém vários exercícios desenvolvidos em **Python**, realizados no âmbito do **Módulo 3 (Programação)**.

Os exercícios abordam conceitos fundamentais da programação, com foco em lógica, estruturas de controlo e manipulação de dados.

---

## 📂 Estrutura do Projeto

O repositório está organizado por ficheiros de exercícios:

### 🧩 Fichas Básicas

* `ex01-Ficha1.py` → Impressão de dados simples

* `ex01-Ficha2.py` → Soma de valores introduzidos

* `ex01-Ficha3.py` → Verificação de nota (>=10)

* `ex01-Ficha4.py` → Simulação de levantamento bancário

* `ex02-Ficha1.py` → Entrada de dados (nome e idade)

* `ex02-Ficha2.py` → Conversão Celsius → Fahrenheit

* `ex02-Ficha3.py` → Verificação de número par

* `ex02-Ficha4.py` → Classificação por idade

* `ex03-Ficha2.py` → Operações matemáticas

* `ex03-Ficha3.py` → Validação de login

* `ex03-Ficha4.py` → Maior de 3 números

* `ex04-Ficha4.py` → Cálculo de descontos

* `ex00-Ficha2.py` → Tipo de variável

* `exp03-Ficha1.py` → Cálculo de idade aproximada

## 🚀 Como Executar

1. Clonar o repositório:

```bash
git clone https://github.com/a14790/3-modulo.git
```

2. Entrar na pasta:

```bash
cd 3-modulo
```

3. Executar qualquer ficheiro:

```bash
python nome_do_ficheiro.py
```
